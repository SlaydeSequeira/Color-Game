Game Help

Game Modes

  • Classic Mode: Guess the displayed color with hex codes or RGB values
  • Colorblind Mode: Guess the color from its RGB values (no visual)
  • Memory Mode: Color is displayed briefly, then you must recreate it
  • Challenge Mode: Race against the clock to guess colors

Scoring

Scores are calculated based on how close your guess is to the target color. A perfect match is 100 points.

Input Types

  • Hex Code: Six-digit hexadecimal value (e.g., FF0000 for red)
  • RGB Values: Red, Green, and Blue values from 0-255
  • Color Picker: Available in some game modes

Tools

Use the color converter to switch between different color formats and help with your guesses.
